Basic Data Types

Author: Steve Russo
Last Updated: August 20, 2015

In programming, a data type describes a variable1 and the memory that the value is contained in that memory. C's arithmetic types (integers and floating points) are covered in the sections below. (For a reference on all data types, check out the C Reference.)

Integers

Below is a table describing integer data types in C. Storage size for each type varies by machine. The ones used for this table are what most modern systems use. (Storage size is in bytes.)

Type Storage Range
char 1 -128 to 127
unsigned char 1 0 to 255
short 2 -32,768 to 32,767
unsigned short 2 0 to 65,535
int 4 -2,147,483,648 to 2,147,483,647
unsigned int 4 0 to 4,294,967,295
long 8 -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
unsigned long 8 0 to 18,446,744,073,709,551,615

Floating Points

Floating point is a different way of saying a decimal number or fractional number. Below is a simple chart explaining the floating point data types in C. (Storage size is in bytes. Precision is the number of decimal places each data type can store.)

Type Storage Precision Range
float 4 6 1.2-38to3.438
double 8 15 2.3-308to1.7308
long double 10 19 3.4-4932to1.14932

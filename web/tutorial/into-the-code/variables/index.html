Variables

Author: Steve Russo
Last Updated: August 20, 2015

A variable is a name given to a section of memory that your program controls. Read on to learn about how to delare and manipulate variables.

Declaring Variables

Delaring a variable is the first thing that must be done before we can use it in any way. There are two major parts that go into a variable declaration: the data type and the name. All variable declarations follow this pattern:
type variable_name;
Below is an example where we delare a integer variable named count. 

int count;
char c;

You could also declare multiple variables in one statement. 

int count, number, id;
double rate, deposit_amount;

Initializing Variables

When a variable is declared, it is assigned the amount of memory mandated by its data type (in this case, int). At this point, the value contained in the variable is whatever was already in that section of memory. If we want to assign a variable an initial value (which is usually a good idea), we must initialize it. Below is an example of initialization of the variable count that is declared as an int. 

int count;
count = 0;

float f;
f = 5.7;

Alternatively, you can declare and initialize a variable in one statement as follows. 

int id = 4253, count = 0;
double deposit_amount = 753.89;
char c = 'C';

Either way has the same result: declare count and it holds the value of 0.

Assigment Operator (=)

The example above is the first time this tutorial uses the equals sign. In C, this is referred to as the assignment operator because it is used to assign values to variables.

Give It a Try

Now that we know how to declare and initialize a variable, lets give it a try in a program. 

#include <stdio.h>

int main() {
	int  count = 0;
	double deposit_amount = 753.89;
	char c = 'C';

	printf("%d\n", count);
	printf("%f\n", deposit_amount);
	printf("%c\n", c);
	return 0;
}

This code declares an integer variable named count and initializes it to 0. Then it prints out the value. Experiment with this program by changing the values of the variables in the program.

Valid Identifiers

Identifier is just another word for a variable or function name. In C, identifiers can only consist of letters and numbers and must start with a letter. (The underscore character (_) is considered a letter.) Variables in C are case-sensitive so a variable named Count is different from a variable named count. There is no established way to name variables or functions in C.

Getting Setup to Code

Author: Steve Russo
Last Updated: September 24, 2015

In order to start coding in C, we need to get our hands on a few resources first.

What You'll Need

There are two things that are essential: a text editor and compiler. A text editor is so that we can easily create and modify our C source code, and a compiler is for turning our source code into instructions that the machine understands. Exactly which text editor and which compiler you use could depend on several factors such as your operating system and personal preferences. Below is a list of Step by Step instruction pages on how to get setup in the environment that is right for you.

Test

Here is a simple C program. Compile this code on your machine to make sure everything works.

#include <stdio.h>

int main() {
	printf("CWhatsGood\n");
	return 0;
}

This program should compile with no errors and no warnings and simply print out the word CWhatsGood. The next section of this tutorial (Program Structure) goes into shallow detail on what this code actually means.

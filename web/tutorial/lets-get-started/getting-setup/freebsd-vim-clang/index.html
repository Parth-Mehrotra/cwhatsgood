Getting Setup With Vim and Clang on FreeBSD

Author: Steve Russo
Last Updated: September 24, 2015

I've been a vim user for a while now and personally can't imagine using another text editor for anything. There is a very steep learning curve with vim, which is why I recommend reading Practical Vim if you want to become a vim pro. In the right hands, vim can be unbelievably efficient.

Installing Vim

Some operating systems built on top of FreeBSD already have vim installed. Just to be safe though, let's open a terminal and run the install command: 

sudo pkg install vim

If vim is already installed, you'll get a message back saying so. If not, this will run through the install after prompting you with how much more disk space will be used.

Clang. Its there.

Clang is the system compiler for FreeBSD. Therefore, it is already there if you are running FreeBSD or any OS built on top of FreeBSD. Now, with vim installed, you are all set to start coding and compling in C!

Relational Operators

Author: Steve Russo
Last Updated: September 3, 2015

Equal To ==, Not Equal To !=

As their names suggest, the equal to and not equal to operators test for equality and inequality (respectively) among two expressions. The == operator will return true if the two expressions are equal, and the != operator will return true if the two expressions are not equal. Below is a simple example in C. 

int n1 = 4;
int n2 = 8;

// check if n1 and n2 are not equal
if (n1 != n2) {
	printf("n1 and n2 have different values\n");
}

// compare the value returned by an
// arithmetic expression to a variable
if (n1 * 2 == n2) {
	printf("n1 * 2 is equal to n2\n");
}

Greater Than >, Less Than <

The greater than operator compares two expressions and returns true if the value of the first expression is greater than the value of the second expression. The less than operator compares two expressions and returns true if the value of the first expression is less than the value of the second expression. Below is a simple example in C. 

int n1 = 4;
int n2 = 8;

// check if n1 is greater than n2
if (n1 > n2) {
	printf("n1 is greater than n2\n");
}

// check if n1 is less than n2
if (n1 < n2) {
	printf("n1 is less than n2\n");
}

... Or Equal To>= <=

The greater than or equal to operator >= checks for equality as well as if the value of the first expression is greater than the value of the second expression. Likewise, the less than or equal to operator <= checks for equality as well as if the value of the first expression is less than the value of the second expression. Below is a simple example in C. 

int n1 = 4;
int n1 = 8;

// check if n1 is greater than or equal to 4
if (n1 >= 4) {
	printf("n1 is greater than or equal to 4\n");
}

// compare the value returned by an
// arithmetic expression to n2
if (6 + 2 <= n2) {
	printf("6 + 2 is less than or equal to n2\n");
}

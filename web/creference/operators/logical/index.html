Logical Operators

Author: Steve Russo
Last Updated: August 22, 2015

Logical AND &&

The logical AND operator requires two operands and returns true when both operands are true. A boolean expression using the logical AND operator short circuts at the first false value. Consider the following code where we check if an integer is inclusively in between 0 and 100: 

// read in an integer
int i;
scanf("%d", &i);

// check if i is greater than or equal to 0
// and that its less than or equal to 100
if (i >= 0 && i <= 100) {
	printf("%d is between 0 and 100\n", i);
} else {
	printf("%d is not between 0 and 100\n", i);
}

First, an integer is read in and stored in i. Since we are checking if i is within a certain range, we have two conditions to be checked and both must evaluates to true. Therefore, we use the logical AND operator which will only return true if both conditions its operating on are true.

Logical OR ||

The logical or operator requires two operands and returns true as long as only one of its operands evaluates to true. Consider the following code where we check if an integer is outside the range of 0 to 100: 

// read in an integer
int i;
scanf("%d", &i);

// check if i is less than 0 or
// that its greater than 100
if (i < 0 || i > 100) {
	printf("%d is outside range of 0 and 100\n", i);
} else {
	printf("%d is in between 0 and 100\n", i);
}

First, an integer is read in and stored in i. Since we are checking if i is outside a certain range, we have two conditions to be checked and one of them must evaluate to true. Therefore, we use the logical OR operator which will return true if one of the conditions its operating on is true. Either i is less than 0 or greater then 100, but it can't be both.

Logical NOT !

The logical NOT operand requires one operand and returns the opposite of that operands logical state. Example: 

int t = 1;
int f = 0;

printf("%d\n", !t); // prints 0
printf("%d\n", !f); // prints 1

Ternary ?:

The ternary operator(sometimes referred to as the "conditional" operator) requires three operands. It is typically used for decision making when assigning values to variables. The syntax for the ternary operator is:
(condition) ? condition_true : condition_false
. The code below illustrates how a value would be assigned to a variable using an if statement. Assume n has some unknown integer value. 

int r;
if (n != 0) {
	r = 100 / n;
} else {
	r = 0;
}

This code can be simplified using the ternary operator as shown below: 

int r = (n != 0) ? 100 / n : 0;

The code in both the examples above aims to divide 100 by an integer with an unknown value. In order to prevent from dividing by 0 (which causes undefined behavior), there is a check that i is not 0 before any division. The code in both examples produce the same exact results. The ternary operator simplifies the code.

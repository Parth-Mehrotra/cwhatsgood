Arithmetic Operators

Author: Steve Russo
Last Updated: August 20, 2015

Binary Arithmetic Operators

Binary operators are operators that require two operands. The binary arithmetic operators in C are addition(+), subtraction(-), multiplication(*), division(/), and modulus(%). Below are very basic examples of the addition, subtraction, multiplication, and division operators. 

// a will have a value of 10
int a = 8 + 2;
// b will have a value of 6
int b = 8 - 2;
// c will have a value of 16
int c = 8 * 2;
// d will have a value of 4
int d = 8 / 2;

Modulus Operator %

The modulus operator returns the remainder of integer division. For example, this could be really useful when checking if a number is even or odd. 

// If result is 0, then some_integer is even.
// If result is 1, then some_integer is odd.
int result = some_integer % 2;

Unary Arithmetic Operators

Unary operators are operators that require only one operand. The arithmetic unary operators in C are the incrememnt operator(++) and the decrement operator(--). These operators can either be placed before the operand(prefix) or after the operand(postfix).

Prefix

When an increment or decrement operator is placed before the operand, it is called a prefix increment or prefix decrement operator. A prefix operator first increments or decrements the operand before it is used in its current expression. For example: 

int i = 0;
printf("%d\n", ++i);

The above code outputs 1 since the prefix operator says increment i by 1 before it used in the current expression.

Postfix

When an increment or decrement operator is placed after the operand, it is called a postfix increment or postfix decrement operator. A postfix operator first uses the operand in its current expression and then increments or decrements it. For example: 

int i = 0;
printf("%d\n", i++);

The above code outputs 0 since the postfix operator says use the operand in the current expression and then increment it by 1.

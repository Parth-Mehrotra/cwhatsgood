Bitwise Operators

Author: Steve Russo
Last Updated: August 27, 2015

Bitwise AND &

The bitwise AND operator is a binary operator that performs the AND operation on each of the pairs of bits from both operands. This means the result will only have a 1 at a certain position if both of the operands have a 1 at that same position. Below is an example of raw binary numbers and the result that would come from a bitwise AND operation on them. 

N1: 			0010 0110
N2: 			0110 0101
			---------
(N1 & N2): 	0010 0100

A simple example in C is shown below. 

char n1 = 38;
char n2 = 101;
printf("%d\n", (n1 & n2));

This above code would output 36 which is 0010 0100 in binary.

Bitwise OR |

The bitwise OR operator is a binary operator that performs the OR operation on each of the pairs of bits from both operands. This means the result will have a 1 at the position where either operand has a 1. Below is an example showing raw binary and the result from a bitwise OR operation on them. 

N1: 			0010 0110
N2: 			0110 0101
			---------
(N1 | N2): 	0110 0111

A simple example in C is shown below. 

char n1 = 38;
char n2 = 101;
printf("%d\n", (n1 | n2));

This above code would output 103 which is 0110 0111 in binary.

Bitwise XOR ^

The bitwise XOR operator (sometimes referred to as the "bitwise exclusive OR operator") is a binary operator that performs the XOR operation on each of the pairs of bits from both operands. This means the result will have a 1 at the positions where the pairs of bits from each operand are different. Below is an example showing raw binary and the result from a bitwise XOR operation on them. 

N1: 			0010 0110
N2: 			0110 0101
			---------
(N1 ^ N2): 	0100 0011

A simple example in C is shown below. 

char n1 = 38;
char n2 = 101;
printf("%d\n", (n1 ^ n2));

This above code would output 67 which is 0100 0011 in binary.

Bitwise NOT ~

The bitwise NOT operator (often times referred to as the "One's Compliment operator") is a unary operator that performs the NOT operation on each bit. This means if the bit is a 1, it becomes a 0. If the bit is a 0, it becomes a 1. The example below uses raw binary to show an operand and the result when the bitwise NOT operation is performed on that operand. 

N1: 		0010 0110
		---------
(~N1): 	1101 1001

A simple example in C is shown below. 

char n1 = 38;
printf("%d\n", ~n1);

The code above would output -39 which is 1101 1001. (Since n1 is a signed char, the high order bit is used to determine if its positive or negative.)

Bitwise Right Shift >>

The bitwise right shift operator is a binary operator that will shift the first operand's bits to the right the number of times specified by the second operand. The example below uses raw binary to show an operand that will have its bits shifted to the right one time. 

N1: 			0010 0110
			---------
(~N1 >> 1): 	0001 0011

A simple example in C is shown below. 

char n1 = 38;
printf("%d\n", (n1 >> 1));

The code above would output 19 which is 0001 0011.

Bitwise Left Shift <<

The bitwise left shift operator is a binary operator that will shift the first operand's bits to the left the number of times specified by the second operand. The example below uses raw binary to show an operand that will have its bits shifted to the left one time. 

N1: 			0010 0110
			---------
(~N1 << 1): 	0100 1100

A simple example in C is shown below. 

char n1 = 38;
printf("%d\n", (n1 >> 1));

The code above would output 76 which is 0100 1100.

Void

Author: Steve Russo
Last Updated: August 29, 2015

The keyword void is used to represent the absence of a data type. In C, this can be used in three scenarios (covered below).

Function Return Value

Functions that do not return a value can be declared as void. For example: 

void has_no_return_val() {
	printf("Printing from void function\n");
}

To return from a void function at any point, simply use return; since there is no value being returned.

Void Pointers

As stated above, void represents the absence of a data type. Therefore, a void pointer can be thought of as "a pointer of some type." Void pointers can be casted to any type. Below is an example of how a void pointer can hold a pointer of any type. 

int i = 25;
char c = 'S';
double d = 87.96;

void* vptr;
vptr = &i; // vptr now has address of i
vptr = &c; // vptr now has address of c
vptr = &d; // vptr now has address of d

Function Argument

If a function does not accept any arguments, you can place the keyword void where an argument list would be to show that the function takes no arguments. For example: 

void foo() {
	printf("Printing from void function\n");
}

void bar() {
	foo(void);
}

This is a feature that was used more often when C was first created than today. However, it is still valid and you still could see it every now and then.

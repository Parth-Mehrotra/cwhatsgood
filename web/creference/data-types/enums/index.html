Enumerations

Author: Steve Russo
Last Updated: September 8, 2015

An enumeration, or enum for short, is a set of integer constants. The integer constants declared in the enum are called enumerators. A enum must declare at least one enumerator. However, specifying a name for the enum is optional as well as naming any identifiers after the enum declaration. The basic syntax for declaring an enum is:

enum enum-name[optional] {
	enumerators
} identifiers[optional];
For example, an enum could be named and used as follows: 

enum day_of_week {
	SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
};

enum day_of_week day = TUESDAY;

Or, the same could be accomplished with:

enum {
	SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
} day;

day = TUESDAY;

Changing the default values

By default, the first enumerator in an enum has a value of 0, and every enumerator after that has the value of the previous enumerator plus 1. This can be changed by explicitly giving values to each enumerator. For example, let's say we want the enumerators in the day of the week enum to contain the day number each is in the week. 

enum day_of_week {
	SUNDAY = 1, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
};

Now the value of each enumerator is increased by 1 from the previous example since the initial enumerator value is 1.

Using enum in struct

It is very easy to use an enum to store a value of a data member in a struct. Let's say there is a date struct that stores the day of the month, month, and year. The month can only be one of twelve possibilities and would be most efficiently stored as an integer value. This is the perfect situation for using an enum. 

#include <stdio.h>

struct date {
	// Declare enum and month identifier
	// right in the struct
	enum {
		JANUARY = 1, FEBRUARY, MARCH, APRIL, MAY, JUNE, JULY,
		AUGUST, SEPTEMBER, OCTOBER, NOVEMBER, DECEMBER
	} month;

	int day_of_month;
	int year;
} d;

int main() {
	d.month = SEPTEMBER; // enum constants visible in declared scope
	d.day_of_month = 7;
	d.year = 2015;

	printf("The date is %d/%d/%d\n", d.month, d.day_of_month, d.year);
	return 0;
}

Output:

The date is 9/7/2015

Arithmetic Data Types

Author: Steve Russo
Last Updated: August 29, 2015

The arithmetic data types in C are integers and floating points (decimals).

To determine the exact size of a data type on your current machine, use:
sizeof(data_type);

Integers

Below is a table showing the standard integer types in C and the range of possible values each can have. The storage size shown for each is the storage size that most C implementations use today. (Storage size is in bytes.)

Type Storage Range
char 1 -27 to 27-1 or -128 to 127
unsigned char 1 0 to 28-1 or 0 to 255
short 2 -215 to 215-1 or -32,768 to 32,767
unsigned short 2 0 to 216-1 or 0 to 65,535
int 4 -231 to 231-1 or -2,147,483,648 to 2,147,483,647
unsigned int 4 0 to 232-1 or 0 to 4,294,967,295
long 8 -263 to 263-1 or -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
unsigned long 8 0 to 264-1 or 0 to 18,446,744,073,709,551,615

Floating Points

Below is a table showing the standard floating point data types in C. (Storage size is in bytes. Precision is number of digits that can be stored after the decimal point.)

Type Storage Precision Range
float 4 6 1.2E-38to3.4E38
double 8 15 2.3E-308to1.7E308
long double 10 19 3.4E-4932to1.1E4932

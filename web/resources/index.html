Books

Online resources are great for learning how to program. However, sometimes nothing beats having a good book in hand. Below are some helpful books for learning more about the C programming language.

More

Browse C Programming Books on Amazon
The C Programming Language
Brian W. Kernighan

Brian W. Kernighan and Dennis Ritche are the guys credited with creating C. This book moves fast if you are looking to learn C, but if offers a unique perspective from the men who created the language.

The Standard C Library
P.J. Plauger

Although this is an old one, I still find myself referencing it often. It breaks down the standard library and has a chapter on each header (assert.h, float.h, etc.). I highly recommend it as a reference for any C programmer.

C Interfaces and Implementations: Techniques for Creating Reusable Software
David R. Hanson

While I do not own this book yet, it is at the top of my wishlist. It covers some very advanced topics and should not be used to learn C from scratch. After I read it for myself, I'll update this review.

Deep C Secrets
Peter van der Linden

This book is more of a "tips and cool tricks" book that assumes much prior knowledge of C. While I have not read this one cover-to-cover, I have skipped around to the various tips and tricks covered within and definitely recommend it for all seasoned C programmers.
